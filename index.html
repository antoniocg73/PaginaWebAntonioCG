Sistema de recomendación de intensificación

Actualmente la ESI presenta cuatro intensificaciones: Computación, Ingeniería de Computadores, Ingeniería del Software y Tecnologías de la Información. Los alumnos cuando llegan a tercero muchas veces no tienen claro que intensificación quieren cursar. Para abordar este problema, desarrollamos una aplicación, donde el alumno responderá un formulario relacionado con las intensificaciones y donde podrá modificar u observar sus calificaciones. El programa analizará la información introducida por el alumno y le ofrecerá una recomendación a partir de esta. Nos gustaría que esta aplicación se usase antes de empezar el segundo cuatrimestre de tercero, y que, por lo tanto, la elección de intensificación fuese una vez se tengan todas las notas del primer cuatrimestre de tercero.

En conclusión, el objetivo de este trabajo es el de realizar un sistema de recomendación de intensificación.

Repositorio del sistema de recomendación

TuriStatSP

TuriStatSp, comprende el desarrollo y entrega de un sistema de información turística capaz de recopilar, analizar y presentar datos sobre la afluencia de turistas en las diferentes regiones de España. Esto incluirá la implementación de dos APIs para la interacción con fuentes de datos externas, la creación de una base de datos, y el desarrollo de una aplicación web interactiva.

La motivación que tuvimos desde el punto de vista de la empresa fue la siguiente:

- Podemos proporcionar a las constructoras datos sobre el flujo de turistas en diferentes destinos y períodos de tiempo. Esta información les permite saber dónde pueden construir alojamientos para obtener un mayor beneficio y conocer los destinos más populares.

- Podemos ayudar a los turistas con estos datos a decidir su destino de viaje, ya que, gracias a los datos recopilados, se puede saber dónde va más gente, que sitios son de interés turístico.
